Liv-ay サービス利用規約

施行日: 2026年7月14日

第1条（目的）

本規約は、CHO-FAM（以下「当社」）が提供するインタラクティブ・ストーリーテリングプラットフォーム「Liv-ay」（以下「本サービス」）の利用に関する、当社と利用者の権利・義務・責任事項を定めることを目的とします。

第2条（定義）

  1. 「本サービス」とは、利用者が分岐型インタラクティブストーリー（ビジュアルノベル形式のゲームコンテンツ）をプレイし、自ら創作・共有し、AI創作ツール（ストーリー構造抽出、画像・BGM・音声生成など）を利用できるプラットフォームをいいます。
  2. 「利用者」とは、本規約に同意し本サービスを利用する会員をいいます。
  3. 「コンテンツ」とは、利用者が本サービスの創作ツールで制作・登録したインタラクティブストーリーおよびそれに含まれるキャラクター、イベント、テキスト、画像、オーディオ等の一切の著作物をいいます。
  4. 「クリエイター」とは、コンテンツを制作・登録し他の利用者に提供する利用者をいいます。
  5. 「コイン」および「ジェム」とは、本サービス内の機能利用に使用される通貨をいい、ジェムは有償で購入できます。
  6. 「キャッシュ」とは、クリエイターがコンテンツ消費に応じて積み立てる精算用通貨（1キャッシュ≒1ウォン）をいいます。

第3条（規約の効力および変更）

  1. 本規約は、本サービス画面への掲示その他の方法により利用者に通知することで効力を生じます。
  2. 当社は関連法令に反しない範囲で本規約を変更でき、変更時は適用日の7日前（利用者に不利な変更は30日前）から告知します。
  3. 変更後の規約に同意しない利用者は、利用を中止し退会できます。

第4条（サービスの提供）

  1. 当社は次のサービスを提供します：インタラクティブストーリーのプレイ（シングル・マルチ）、AI創作ツール（構造抽出、エピソード生成、キャラクター・シーン画像生成、BGM生成、音声合成）、コンテンツの登録・共有およびクリエイター収益精算。
  2. 一部の機能はコインまたはジェムを消費し、消費量は本サービス画面に表示されます。
  3. 当社は運営上・技術上の必要に応じて提供内容を変更できます。

第5条（アカウントおよび利用資格）

  1. 利用者はメールアドレス、電話番号またはソーシャルアカウント（Google、Apple）で登録でき、アカウント情報の管理責任は利用者本人にあります。
  2. 本サービスは満14歳以上が利用できます。
  3. 成人向けと表示されたコンテンツは、成人認証を完了した利用者のみ利用できます。

第6条（有償通貨：コイン・ジェム）

  1. ジェムはアプリ内決済（Google Play、App Store）で購入でき、決済・返金は各アプリマーケットのポリシーに従います。
  2. 未使用ジェムのクーリングオフは関連法令およびアプリマーケットのポリシーに従います。無償で付与されたコイン・ジェムは返金対象外です。
  3. 退会時、残存するコイン・ジェム・キャッシュは消滅し、復元されません。

第7条（コンテンツの権利）

  1. 利用者が創作したコンテンツの著作権は当該利用者に帰属します。
  2. 利用者はコンテンツの登録により、本サービスの運営・表示・広報に必要な範囲で当社にコンテンツの使用（保存、複製、送信、表示、翻訳）ライセンスを付与します。
  3. AI創作ツールで生成された成果物（画像・音楽・音声・テキスト）は、当該コンテンツの一部として本サービス内での利用のために提供されます。
  4. 登録されたコンテンツは、当社の審査（評価）基準を満たした場合に他の利用者へ公開されます。

第8条（著作権保護）

  1. 利用者は、他者の著作権・商標権等の知的財産権を侵害するコンテンツを制作・登録してはなりません。既存の著作物（小説、漫画、ゲーム、映像等）のキャラクター・設定・ストーリーを権利者の許諾なく実質的に借用したコンテンツを含みます。
  2. 当社はコンテンツ登録審査の過程で著作権侵害の可能性を検討でき、侵害が疑われるコンテンツについて公開保留・掲載停止等の措置を行うことができます。
  3. 権利者は本サービス内の通報機能により侵害コンテンツを通報でき、当社は確認後、掲載停止、削除、利用制限等の措置を行います。
  4. 他者の権利を繰り返し侵害した利用者に対し、当社はアカウントの利用を制限できます。

第9条（クリエイター収益精算）

  1. クリエイターは自らのコンテンツで発生したジェム消費について、当社の定める収益率に応じてキャッシュを積み立てます。
  2. キャッシュの現金化は、プロクリエイター等級、最低換金額（10,000キャッシュ）、受取口座の登録など当社の定める条件を満たした場合に申請できます。
  3. 現金化には所得税法に基づく源泉徴収（3.3%）が適用され、手取り額は登録口座へ支払代行サービスを通じて入金されます。
  4. 収益率・精算条件は本サービス画面に掲示し、変更時は事前に告知します。

第10条（禁止行為）

  1. 他者アカウントの盗用、本サービスの不正利用（通貨の不正取得、システム脆弱性の悪用等）
  2. 他者の知的財産権・肖像権・名誉を侵害するコンテンツの制作・登録
  3. 法令または公序良俗に反するコンテンツの制作・登録（成人認証を回避した成人向けコンテンツの提供を含む）
  4. 本サービスの正常な運営を妨げる行為（自動化ツールによる大量リクエスト等）

第11条（サービスの中断および変更）

当社は、システム点検、障害、天災等やむを得ない場合、本サービスの全部または一部を一時中断することができ、事前または事後に告知します。

第12条（契約解除および退会）

  1. 利用者はいつでも本サービス内の退会機能により利用契約を解除できます。
  2. 退会時、個人情報はプライバシーポリシーに従って破棄され、登録コンテンツの取扱いは退会手続き内で案内します。

第13条（責任の制限）

  1. 本サービス内のストーリー進行・画像・音声等はAIにより生成されることがあり、当社はAI生成物の正確性・完全性を保証しません。
  2. 利用者が制作・登録したコンテンツに対する法的責任は当該利用者にあります。
  3. 当社は、当社の故意または重大な過失がない限り、無償で提供されるサービスの利用に関して生じた損害について責任を負いません。

第14条（準拠法および管轄）

本規約は大韓民国の法令に従って解釈され、本サービスの利用に関する紛争は民事訴訟法上の管轄裁判所に提起します。

附則

本規約は2026年7月14日から施行します。
